Phase 7I runner setup

Purpose
Configure commands for PURE and PL-Marker so the harness can run real baselines.

Edit file
scripts/phase7i/runner_cmds.env

Set these variables to absolute commands on your system.
Example values shown using placeholder paths.
Replace with your own interpreter and entrypoints.

RUNNER_PURE_CMD=/opt/conda/envs/pure/bin/python -m pure.run
RUNNER_PLMARKER_CMD=/opt/conda/envs/plmarker/bin/python -m plmarker.run

After configuring, run any of the following
scripts/phase7i/run_bench.sh pure scierc --split=test
scripts/phase7i/run_bench.sh pure biored --split=test
scripts/phase7i/run_bench.sh pl-marker scierc --split=test
scripts/phase7i/run_bench.sh pl-marker biored --split=test

Outputs will appear under
var/receipts/phase7i/<model>/<timestamp>/
and scoreboards are aggregated into
docs/scoreboards/phase7i/

Wrappers (20250909-200046 UTC)

  RUNNER_PURE_CMD=/home/tank/darf-source/scripts/phase7i/wrappers/run_pure.sh
  RUNNER_PLMARKER_CMD=/home/tank/darf-source/scripts/phase7i/wrappers/run_plmarker.sh
